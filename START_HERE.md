## 🔑 What You'll Need to Get

During deployment, you'll need to obtain:

1. **Discord Bot Token** (from Discord Developer Portal)
2. **Discord Application ID** (from Discord Developer Portal)
3. **MongoDB Connection String** (from MongoDB Atlas)
4. **Your Discord User ID** (from Discord)
5. **Your Discord Server ID** (from Discord)

Don't worry - the guides tell you exactly how to get each one!
